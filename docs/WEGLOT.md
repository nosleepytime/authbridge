# Weglot Guide

This project is already prepared for Weglot on the auth pages.

## How it works here

Every page includes:

```html
<script src="/config.js"></script>
<script src="/app.js"></script>
```

`/config.js` is generated by the worker and exposes `WEGLOT_API_KEY` if you set it in Cloudflare secrets.  
Then `app.js` loads Weglot automatically.

## Add your key

```bash
npx wrangler secret put WEGLOT_API_KEY
```

## Result

If a valid Weglot API key is present, the auth UI pages can auto-load Weglot and initialize translation.

## Notes

- This is ideal for quickly translating auth pages.
- You can still customize Weglot options later inside `public/app.js`.
- If you do not set the key, nothing breaks. Translation simply stays disabled.
